# ITPM - Projektmarketing - Instrumente

> [!info] Kurzdefinition
> Sechs Instrumente prägen das IT-Projektmarketing: **Kommunikation, Präsentation/Visualisierung, Argumentation, situative Gesprächsführung, Projektidentität und Networking**. Sie werden situations- und zielgruppenspezifisch eingesetzt.

## Beschreibung

Aus Folie 22:

## Bestandteile / Aufbau

| Instrument | Beschreibung |
|---|---|
| **Kommunikation** | Die richtigen Dinge zum richtigen Zeitpunkt, im richtigen Maß kommunizieren. |
| **Präsentation / Visualisierung** | Ideenpräsentation, Ergebnispräsentation usw. |
| **Argumentation** | Unsicherheiten beseitigen mithilfe von überprüfbaren Tatsachen, Berufung auf Experten, … |
| **Situative Gesprächsführung** | An Kontext und Gesprächspartner angepasste Gesprächsführung. |
| **Projektidentität** | Projektname, Projektlogo, … |
| **Networking** | Aufbau und Pflege relevanter Beziehungen. |
