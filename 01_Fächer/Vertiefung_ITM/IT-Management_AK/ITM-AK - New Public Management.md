# ITM-AK - New Public Management

> [!info] Kurzdefinition
> New Public Management (NPM) ist ein Reform-Ansatz, der privatwirtschaftliche Steuerungslogik (Effizienz, Outputmessung, Wettbewerb) in öffentliche Einrichtungen überträgt. Im Gesundheitsbereich ist es ein wichtiger Treiber für obligatorische Qualitätsmessung und leistungsorientierte Vergütung.

## Beschreibung

Folie 59 (Value-based Healthcare) listet **New Public Management** als einen der Treiber neben:
- Neues Konzept für die Gesundheitsversorgung (Qualität).
- Fokus auf Wertschöpfung und Kosten.
- Definition: „value is defined as the health outcomes achieved per dollar spent" (Porter, 2008, S. 503).
- **New Public Management.**
- **(Obligatorische) Qualitätsmessung.**
- **Bezahlung nach Volumen vs. Bezahlung nach Leistung.**

Damit verortet die VL NPM als methodisches Dach, unter dem Value-based Healthcare im öffentlichen Gesundheitssektor operiert.

## Bestandteile / Aufbau

NPM bringt typischerweise (eigene Antwort, basierend auf Standardwissen, da Folie nur den Begriff nennt):

| Element | Inhalt |
|---|---|
| **Outputmessung** | Klare Leistungs- und Outcome-Indikatoren statt reiner Inputkontrolle |
| **Wettbewerbselemente** | Quasi-Märkte, Benchmarking, Wahlfreiheit |
| **Ergebnisorientierte Steuerung** | Verträge mit Zielvereinbarungen statt hierarchischer Anweisung |
| **Dezentralisierung** | Verlagerung von Entscheidungen in operative Einheiten |
| **Externe Verantwortlichkeit** | Veröffentlichung von Performance-Daten |

Im österreichischen Gesundheitswesen findet sich NPM-Logik in:
- A-IQI (verpflichtende Qualitätsindikatoren mit öffentlichem Reporting).
- LKF (leistungsorientierte Krankenanstaltenfinanzierung).
- Trend zu Value-based Healthcare (Outcomes pro Euro).

## Abgrenzung

- **NPM ≠ Privatisierung:** NPM bringt privatwirtschaftliche Logik in *öffentliche* Einrichtungen, ohne sie zu privatisieren.
- **NPM ≠ Value-based Healthcare:** VBHC ist ein Inhalt; NPM ist die Reform-Klammer, in der VBHC im öffentlichen Sektor entstehen kann.
- **NPM ≠ Lean / Six Sigma:** Lean/Six Sigma sind Methoden der operativen Effizienz; NPM ist eine Verwaltungs-/Steuerungs­doktrin.

## Prüfungsrelevanz

**Typische Definitionsfrage:** „Was bezeichnet New Public Management?"

**Anwendungsfrage:** „Welche Beispiele für NPM-Einflüsse finden sich im österreichischen Gesundheitswesen?" Antwort: A-IQI, LKF, Outcome-Orientierung.

**Diskussionsfrage:** Welche Vor- und Nachteile bringt NPM im Gesundheitswesen?
- **Vorteile:** Transparenz, Effizienzdruck, Vergleichbarkeit, Patient:innenorientierung.
- **Nachteile:** „Ökonomisierung" der Versorgung, Risiko von Risikoselektion, Reduktion auf messbare Indikatoren, Demotivation der Mitarbeitenden durch Kontrolllogik.
- Bezug zu Pflueger 2015 (PDF 6) – kritische Reflexion der Quality-Accounting-Logik.

**Mögliche Anschlussfragen:**
- Wie hängt NPM mit der Balanced Scorecard zusammen (sie wird oft zusammen eingeführt)?
- Welche Reformen markieren NPM in Österreich?
- Inwiefern ist NPM Voraussetzung für VBHC im öffentlichen Sektor?
