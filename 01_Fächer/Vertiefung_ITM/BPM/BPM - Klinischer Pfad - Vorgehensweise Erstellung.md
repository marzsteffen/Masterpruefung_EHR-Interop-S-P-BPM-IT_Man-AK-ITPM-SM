# BPM - Klinischer Pfad - Vorgehensweise Erstellung

> [!info] Kurzdefinition
> Praxisnaher Vier-Phasen-Workflow für die Pfad-Erstellung: **IST-Prozess → SOLL-Prozess → Implementierung → Evaluierung/KVP-Prozess**. Jede Phase hat klare Aktivitäten und Endpunkte. Diese Vorgehensweise verbindet die ÖNORM-K-1930-Anforderungen mit der allgemeinen 4-Schritte-Methode aus dem PzM-Kompakt.

## Beschreibung

Folie schluss­seite (Folie 46 / unbenannt – das letzte Schaubild der VL).

### Phase 1: IST-Prozess

| Aktivität | Inhalt |
|---|---|
| Qualitative Interviews | mit beteiligten Mitarbeiter:innen |
| Beobachtungen | direkt im Stations­alltag |
| Workshops | mit dem multiprofessionellen Team |
| An der Abteilung verwendete Leit- und Richtlinien | sammeln + auswerten |
| Ev. Analyse von Krankenakten | retrospektiv |
| Literatursuche | Stand der Evidenz |
| **Endpunkt** | **Modellierung des IST-Prozesses** |

### Phase 2: SOLL-Prozess

| Aktivität | Inhalt |
|---|---|
| Zusammen­tragen der Änderungs­wünsche | aus den IST-Prozess-Interviews |
| Kombination mit Erkenntnissen | aus historischen Krankenakten |
| Workshops zur Erstellung | eines SOLL-Prozesses |
| **Endpunkt** | **Modellierung des SOLL-Prozesses** |

### Phase 3: Implementierung

| Aktivität | Inhalt |
|---|---|
| Erstellung von Schulungs­dokumenten | berufsgruppen­spezifisch |
| Mitarbeiter­schulungen | aller Beteiligten |
| Zu Beginn: Rückspiegelung der Ergebnisse im Wochentakt | hohe Frequenz in der Anlaufphase |
| Erstellung von prozess­relevanten Dokumenten | KIS-Module, Checklisten |
| Erstellung von Infomaterial für Zuweiser | Hausärzt:innen, andere Kliniken |
| Information von Zuweisern | aktiv informieren |
| **Endpunkt** | **Formulierung des neuen Pfades als Richtlinie für die Mitarbeiter:innen** |

### Phase 4: Evaluierung / KVP-Prozess

| Aktivität | Inhalt |
|---|---|
| Rückspiegelung der Ergebnisse | regelmäßige Reports |
| Verwendung der Ergebnisse für Marketing-Zwecke | externe Kommunikation der Pfad-Wirkung |
| Überführung in einen KVP-Prozess | Daueraufgabe statt Projekt­abschluss |

## Bestandteile / Aufbau

```
1. IST-Prozess
   Interviews + Beobachtungen + Workshops + Leitlinien-Review + Krankenakten + Literatur
   → IST-Modellierung

2. SOLL-Prozess
   Änderungswünsche + Krankenakten-Erkenntnisse + SOLL-Workshops
   → SOLL-Modellierung

3. Implementierung
   Schulungsdokumente + Schulungen + Wöchentliches Feedback
   + KIS-Dokumente + Zuweiser-Info
   → Pfad ist freigegebene Richtlinie

4. Evaluierung / KVP
   Rückspiegelung + Marketing + Daueraufgabe KVP
```

### Bezug zur ÖNORM K 1930 und zur Eight-Step-Methode

| Phase | ÖNORM K 1930 | Eight-Step (Lodewijckx) |
|---|---|---|
| IST-Prozess | (nicht explizit als Phase, aber in §4.1 Anmerkung „bestehende Pfade berücksichtigen") | Schritte 1+2 (Definition + Recherche) |
| SOLL-Prozess | §4.1 Pfadentwicklung | Schritte 3–6 (Delphi, Auswahl, Bündelung, Detail) |
| Implementierung | §4.3 + §4.3.1 | Schritt 8 (Pilotierung) |
| Evaluierung / KVP | §4.4 + §4.5 | Schritt 7 (Indikatoren) + KVP |

→ Die Vorgehensweise ist eine pragmatische Synthese.

## Beispiel

Hernien-Pfad-Erstellung in einer Klinik:
- **Phase 1 IST**: 3 Monate Interviews mit Chirurgie/Anästhesie/Pflege; teilnehmende Beobachtung an 2 Stationen; Auswertung von 20 retrospektiven Hernien-Fällen; Literatursuche zu HerniaSurge / EH-Society. Ergebnis: BPMN-modellierter IST-Pfad.
- **Phase 2 SOLL**: 3 Workshops zur Erstellung; Integration von Choosing-Wisely-Empfehlungen zur Reduktion unnötiger präop. Tests; Konsens­findung mit Delphi-light-Methode. Ergebnis: SOLL-Pfad.
- **Phase 3 Implementierung**: 4-stündige Schulungen für Chirurgie/Anästhesie/Pflege; KIS-Modul „Hernien-Pfad" mit Pflichtfeldern; wöchentliche Pfad-Reviews in den ersten 8 Wochen; Brief an alle einweisenden Hausärzt:innen.
- **Phase 4 KVP**: monatliche Auswertung Komplikationsrate / Adhärenz / PROMs; Ergebnis-Marketing in Kliniknews; Pfad-Update bei neuer HerniaSurge-Aktualisierung.

## Abgrenzung

- **Vorgehensweise ≠ ÖNORM K 1930.** ÖNORM ist *Anforderungs­katalog*; Vorgehensweise ist *praktischer Workflow*. Beide gehen Hand in Hand.
- **Vorgehensweise ≠ Eight-Step.** Eight-Step ist *methodisch dichter* (Delphi, Indikatoren); die Vorgehensweise ist *organisations­näher* (IST-Aufnahme, Schulungen, Marketing).
- **Phasen sind nicht strikt sequenziell.** Iterationen zwischen IST und SOLL sind häufig (z. B. erst SOLL skizzieren, dann erkennen, dass IST-Verständnis lückenhaft war).

## Prüfungsrelevanz

**Typische Definitionsfrage:** „Welche vier Phasen umfasst die praxisnahe Vorgehensweise zur Pfad-Erstellung?"

**Anwendungsfrage:** „Welche Aktivitäten gehören zur IST-Aufnahme?" – Interviews, Beobachtungen, Workshops, Leitlinien-Review, Krankenakten, Literatursuche.

**Diskussionsfrage:** Warum ist die wöchentliche Rückspiegelung in der Anlauf­phase so wichtig? – Frühe Probleme werden früh erkannt; das Vertrauen der Anwender:innen wird gestärkt; Verbesserungs­vorschläge bleiben aktuell statt zu „verstauben". Bezug zu [[BPM - Bewusstseinskreis]].

**Mögliche Anschlussfragen:**
- Was ist der Endpunkt jeder Phase?
- Welche Rolle spielen Zuweiser?
- Wie hängt die Vorgehensweise mit der ÖNORM zusammen?
- Wie hängt sie mit der Eight-Step-Methode zusammen?
