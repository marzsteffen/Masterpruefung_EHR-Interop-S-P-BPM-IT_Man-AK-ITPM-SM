# ITM-AK - Bilanz

> [!info] Kurzdefinition
> Die Bilanz (engl. balance sheet) bildet Vermögenswerte (Aktiva), Verbindlichkeiten und Eigenkapital eines Unternehmens zu einem Stichtag (in der Regel Jahresende) ab. Es gilt: Eigenkapital = Aktiva − Passiva.

## Beschreibung

Über die Folien 58–63 wird die Bilanz schrittweise erläutert:

### Bilanz I (Folie 58)
- **Vermögenswerte:** Barmittel, Vorräte, Sachanlagen und sonstige Investitionen.
- **Passiva & Eigenkapital:** Eigenkapital der Shareholder, lang- und kurzfristige Verbindlichkeiten.
- **Eigenkapital = Aktiva − Passiva.**

### Bilanz II (Folie 59)
- Abbildung der Vermögenswerte, Verbindlichkeiten und des Eigenkapitals eines Unternehmens.
- Drei Leitfragen: „Was gehört dem Unternehmen?", „Was ist das Unternehmen schuldig?", „Was haben die Anteilseigner?"
- Beschreibt die finanzielle Situation **am Ende eines jeden Rechnungszeitraums**.

### Bilanz III (Folie 60) – Aktiva-Untergliederung

| Umlaufvermögen | Langfristige Vermögenswerte |
|---|---|
| Bargeld und marktgängige Wertpapiere (kurzfristig, risikoarm) | Immobilien, Ausrüstung, Anlagen, Fahrzeuge, … |
| Debitoren | Immaterielle Vermögenswerte |
| Vorräte | |
| Sonstiges Umlaufvermögen | |

### Vermögensübersicht IV (Folie 61) – Passiva-Untergliederung

| Verbindlichkeiten | Eigenkapital |
|---|---|
| Kurzfristige Verbindlichkeiten (z. B. Lieferantenkredite) | Gewöhnliches Eigenkapital |
| Langfristige Verbindlichkeiten (Schulden, Leasing, gestundete Steuern) | Sonstiges Eigenkapital (Dividenden, …) |
| | Einbehaltene Gewinne |

### Bilanz in Kurzform (Folie 63)

- Wichtiger Teil des Finanzmanagements.
- Stellt den **Status quo zu einem bestimmten Datum** dar (in der Regel Jahresende).
- IFRS liefert Richtlinien, wie Positionen zu **bewerten** und zu bilanzieren sind (z. B. Vorräte, Leasing, Sachanlagen) → Buchung des „richtigen" Wertes.

## Bestandteile / Aufbau

Bilanzgleichung:

```
Aktiva (Vermögen) = Passiva (Schulden) + Eigenkapital
            ↓
   Eigenkapital = Aktiva − Passiva
```

Visualisiert als Zwei-Spalten-Aufstellung:

```
| AKTIVA                  |  PASSIVA              |
|-------------------------|-----------------------|
| Umlaufvermögen          |  Kurzfristige Verb.   |
|   - Bargeld             |  Langfristige Verb.   |
|   - Forderungen         |  Eigenkapital         |
|   - Vorräte             |    - Stammkapital     |
| Anlagevermögen          |    - Rücklagen        |
|   - Sachanlagen         |    - Einbehalt. Gew.  |
|   - Immaterielles       |                       |
```

## Abgrenzung

- **Bilanz ≠ GuV:** Bilanz ist Stichtagsbetrachtung, GuV ist Periodenbetrachtung (siehe [[ITM-AK - Gewinn- und Verlustrechnung]]).
- **Buchwert ≠ Marktwert:** Die Bilanz zeigt Buchwerte (nach IFRS bewertet), nicht Marktwerte (Börsenwert).
- **Aktiva ≠ Vermögen im umgangssprachlichen Sinn:** Aktiva sind die bilanzielle Darstellung; das tatsächliche „Vermögen" eines Unternehmens umfasst auch nicht-bilanzierte Werte (Marke, Humankapital).

## Prüfungsrelevanz

**Typische Definitionsfrage:** „Beschreiben Sie die Elemente einer Bilanz." (Frage zur Wiederholung Folie 83)

**Anwendungsfrage:** „Wo wird ein neu gekauftes MRT-Gerät in einer Krankenhaus-Bilanz erfasst?" Antwort: Sachanlagen (Anlagevermögen), Gegenkonto je nach Finanzierung (Kasse / Verbindlichkeit).

**Diskussionsfrage:** Wie wirken sich IFRS-Bewertungsregeln auf die Aussagekraft der Bilanz aus? Diskussion: IFRS verlangt teilweise Fair-Value-Bewertung (z. B. Finanzinstrumente), was die Bilanz volatiler macht. Sachanlagen können nach Anschaffungskosten oder Neubewertung bilanziert werden.

**Mögliche Anschlussfragen:**
- Wie hängt die Bilanz mit der GuV (einbehaltene Gewinne) zusammen?
- Was bedeutet das Working Capital (Umlaufvermögen − kurzfristige Verbindlichkeiten)?
- Welche Bilanzkennzahlen gibt es (Debt-equity ratio, Current ratio etc.)?
