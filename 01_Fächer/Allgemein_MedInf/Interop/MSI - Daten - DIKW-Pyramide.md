# MSI - Daten - DIKW-Pyramide

> [!info] Kurzdefinition
> Hierarchisches Modell, das Zeichen → Daten → Information → Wissen → Aktion über die Stufen Erkennen, Verstehen im Kontext, In-Beziehung-setzen, Anwenden verbindet. In der Vorlesung als Rahmen, um zu zeigen, was Standards/Terminologien zwischen Daten und Aktion leisten müssen.

## Beschreibung

Folie 13 zeigt die Pyramide mit Quellenverweis „nach Aamodt und Nygård, 1995" als Teil einer IKS-Folie zu „Was ist Wissen?":

- **Zeichen** (z. B. „H, K, R, 0, 0, 0, 5, €")
- → Erkennen → **Daten** (z. B. „HR K 5000 €")
- → Verstehen im Kontext → **Information** (z. B. „Hr. K. hat 5.000 € überwiesen.")
- → In Beziehung setzen → **Wissen** („Ungewöhnlich hoher Betrag für Hr. K. (bisher nie mehr als 1.000 €)") – „Weisheit" als Variante darüber
- → Anwenden → **Aktion** („Überweisung stoppen. Hr. K. kontaktieren.")

Übergänge zwischen Information und Wissen werden in der Folie über „Training / Lernen" beschriftet. Die Folie endet mit dem Auftrag „Beispiel aus dem Gesundheitssystem?" – also als Diskussionsanker für die LV.

## Bestandteile / Aufbau

Fünf Stufen plus die Übergangsoperationen:

| Stufe | Beispiel (Folie) | Operation zur nächsten Stufe |
|---|---|---|
| Zeichen | H, K, R, 0, 0, 0, 5, € | Erkennen |
| Daten | HR K 5000 € | Verstehen im Kontext |
| Information | Hr. K. hat 5.000 € überwiesen. | In Beziehung setzen |
| Wissen | Ungewöhnlich hoher Betrag für Hr. K. | Anwenden |
| Aktion | Überweisung stoppen, Hr. K. kontaktieren | – |

## Beispiel

Eigenes Gesundheits-Beispiel wird in der Folie nicht ausgeführt (ist Diskussionsfrage). Im Vault-Kontext: Pulswert „59" → Daten („Puls 59/min") → Information („bradykard") → Wissen („für Hr. K. unauffällig wegen Sport") → Aktion („keine Therapie").

## Abgrenzung

- **Daten ≠ Information**: Daten ohne Kontext sind nicht informativ.
- **Information ≠ Wissen**: Erst durch Bezug zu anderen Informationen (Verlauf, Vergleich) entsteht Wissen.

## Prüfungsrelevanz

**Typische Definitionsfrage:** Welche Stufen umfasst die DIKW-Pyramide laut Vorlesung, und welche Operationen verbinden sie?

**Anwendungsfrage:** Geben Sie ein eHealth-Beispiel pro Stufe.

**Diskussionsfrage:** An welchen Übergängen setzt semantische Interoperabilität an? (Antwort: vor allem zwischen Daten und Information, und zwischen Information und Wissen – Standards/Terminologien sichern den Bedeutungstransfer.)

**Mögliche Anschlussfragen:**
- Wo platzieren Sie ML/KI in dieser Pyramide?
- Welche Konsequenz hat schlechte Datenqualität für die oberen Stufen?
- Was bedeutet „Weisheit" als Erweiterung von Wissen?
