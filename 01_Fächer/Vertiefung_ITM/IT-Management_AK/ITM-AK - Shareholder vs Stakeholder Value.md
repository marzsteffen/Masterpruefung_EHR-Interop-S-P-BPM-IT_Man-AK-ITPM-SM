# ITM-AK - Shareholder vs Stakeholder Value

> [!info] Kurzdefinition
> Zwei Perspektiven auf den Wert eines Unternehmens: Shareholder Value misst den Wert für Aktionäre (finanziell, Aktien); Stakeholder Value betrachtet den Wert für alle Interessengruppen (Gesellschaft, Umwelt, Mitarbeiter:innen, Kund:innen).

## Beschreibung

Folie 48 stellt beide Konzepte gegenüber:

### Wert für die Aktionäre (Shareholder Value)
- **Finanzielle Laufzeit.**
- **Wie viel ist das Unternehmen in Geld wert?**
- **Kauf von Aktien → Eigentum an einem Vermögenswert.**

### Wert für die Interessengruppen (Stakeholder Value)
- **Breitere Perspektive.**
- **Einschließlich interner und externer Interessengruppen.**
- z. B. **Gesellschaft, Umwelt, Mitarbeiter, Kunden.**

Die Folie nennt die Frage „Was ist der Wert eines Unternehmens?" als „viel diskutiertes Thema" (Folie 47–48).

## Bestandteile / Aufbau

| Perspektive | Adressat | Maßstab |
|---|---|---|
| Shareholder Value | Aktionäre / Eigentümer | Aktienkurs, Dividenden, Gewinn |
| Stakeholder Value | Mitarbeiter, Kunden, Gesellschaft, Umwelt | breiter Wertbegriff (auch nicht-monetär) |

## Beispiel

Im PDF nicht ausgearbeitet. Folie 24 (Mediclinic Vision: „TO ENHANCE THE QUALITY OF LIFE") suggeriert Stakeholder-orientierte Sichtweise (Quality of Life für Patient:innen).

## Abgrenzung

- **Shareholder Value ≠ Profit:** Profit ist eine Periodengröße; Shareholder Value bezieht sich auf den Marktwert (Aktienkurs × Anzahl Aktien) plus zukünftige Cashflows.
- **Stakeholder Value ≠ ESG / CSR:** ESG (Environmental, Social, Governance) und CSR (Corporate Social Responsibility) sind operative Konzepte zur Umsetzung von Stakeholder-Orientierung – aber nicht synonym.
- Im PDF werden Shareholder und Stakeholder gegenübergestellt, nicht als sich ausschließend dargestellt – beide können kombiniert werden („shared value", Porter/Kramer 2011 – nicht in dieser VL).

## Prüfungsrelevanz

**Typische Definitionsfrage:** „Was ist der Unterschied zwischen Stakeholder- und Shareholder-Value?" (= Frage zur Wiederholung Folie 52)

**Anwendungsfrage:** „Welche Sicht ist im Gesundheitswesen besonders relevant?" Diskussion: öffentlich-finanzierte Krankenhäuser sind stark Stakeholder-orientiert (Versorgungsauftrag, gesellschaftlicher Wert); private Krankenhauskonzerne unterliegen zusätzlich Shareholder-Druck.

**Diskussionsfrage:** Können Shareholder Value und Stakeholder Value im Konflikt stehen? Beispiele:
- Investitionen in Mitarbeiter-Weiterbildung erhöhen Stakeholder Value, kosten kurzfristig Shareholder Value.
- Aggressive Kostenreduktion erhöht Shareholder Value kurzfristig, kann Patientensicherheit (Stakeholder) gefährden.
- Long-term Shareholder Value lässt sich oft mit Stakeholder Value vereinbaren – kurzfristiger Shareholder Value oft nicht.

**Mögliche Anschlussfragen:**
- Wie messen Sie Shareholder Value, wie Stakeholder Value?
- Welche Rolle spielt ESG-Reporting in dieser Diskussion?
- Wie wirkt sich die Trägerstruktur (öffentlich/privat/gemeinnützig) auf die Wertdiskussion aus?
